# Design System Documentation

## Theme Overview

This document outlines the core visual characteristics of our design system, derived directly from the theme's technical specifications.

---

## Brand Identity

### Color Palette

Our brand utilizes a vibrant and modern color palette designed to create an engaging user experience.

- **Primary Color**: `#0082fd` - A bright, active blue used for key interactive elements and primary calls to action.
- **Secondary Color**: `#5071d5` - A supportive blue-violet, complementing the primary for less prominent UI components and secondary actions.
- **Tertiary Color**: `#a459b5` - An accent purple, adding vibrancy for highlights, badges, or decorative details.
- **Neutral Color**: `#6a769e` - A soft, cool grey-blue serving as a foundational color for backgrounds and non-chromatic elements.

The overall color mode is set to **light**, ensuring a bright and airy interface.

### Typography

Our typographic system is built for clarity and consistency across all text elements.

- **Headline Font**: `Inter` - Used for all headings, providing a strong and legible presence.
- **Body Font**: `Inter` - Applied to all body text, ensuring readability and consistency.
- **Label Font**: `Inter` - Dedicated to labels and small text elements, maintaining brand uniformity.

### Shape & Form

The visual styling of UI elements emphasizes a balanced and approachable aesthetic.

- **Roundedness**: `2` - Elements feature **moderate** rounded corners, providing a friendly and modern feel without being overly soft.

### Spacing & Layout

The spacing system ensures a comfortable and organized layout, promoting readability and ease of interaction.

- **Spacing**: `2` - A **normal** level of whitespace is applied, offering a good balance between compactness and an open, spacious feel.
